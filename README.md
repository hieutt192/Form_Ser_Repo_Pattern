### Run Repo

``` bash
# use vuejs 
$ npm install
```

``` bash
# dev server with hot reload 
npm run dev
```

``` bash
# Dev server running on [http://127.0.0.1:8000].
php artisan serve
```

``` bash
# build composer larvel in file composer.json 
composer update
```

``` bash
# create jwt secret
php artisan jwt:secret
```
